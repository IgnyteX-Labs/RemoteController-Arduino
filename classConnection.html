Detailed Description

This abstract class interfaces between the RemoteController and any way of transmitting the data for the RemoteController. E.g. WiFi and RF24.

◆ getMaxPackageSize()

virtual size_t Connection::getMaxPackageSize ( )
pure virtual

The maximum size, in byte, that can be send in one package i.e. with one Connection::write() call.

Warning
If this is smaller than the size of one command (i.e. 2 bytes (identifer) + 5 bytes (command) = 7 bytes) you will encounter unexpected problems.

Implemented in RF24Connection.
